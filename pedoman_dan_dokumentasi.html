Panduan Penggunaan

Langkah-langkah mudah untuk memulai perjalanan pembelajaran Anda

1. Registrasi Akun

Daftarkan akun baru dengan mengisi data diri yang valid. Pastikan menggunakan email aktif untuk verifikasi.

Tip: Gunakan password yang kuat dengan kombinasi huruf, angka, dan simbol untuk keamanan maksimal.
2. Lengkapi Profil

Update informasi profil Anda untuk mendapatkan rekomendasi konten yang lebih personal dan relevan.

3. Jelajahi Dashboard

Kunjungi dashboard utama untuk melihat progress belajar, quick actions, dan statistik pembelajaran Anda.

4. Akses Materi Pembelajaran

Pilih materi yang ingin dipelajari dari berbagai kategori kesehatan remaja yang tersedia.

  • Materi PDF untuk pembelajaran mendalam
  • Video edukasi untuk visual learning
  • Konten interaktif untuk engagement
5. Ikuti Progress Belajar

Pantau kemajuan belajar Anda melalui dashboard yang menampilkan statistik lengkap dan pencapaian.

6. Bergabung dengan Komunitas

Berpartisipasi dalam forum diskusi untuk berbagi pengalaman dan belajar dari sesama pengguna.

7. Konsultasi dengan Ahli

Jadwalkan sesi konsultasi langsung dengan ahli kesehatan melalui fitur Zoom Meeting.

Informasi Teknis

Detail teknis platform untuk developer dan administrator

Teknologi yang Digunakan
  • Backend: Laravel 10.x Framework
  • Frontend: Bootstrap 5, JavaScript ES6+
  • Database: MySQL 8.0+
  • Server: Apache/Nginx dengan PHP 8.1+
  • Cache: Redis/File Cache
Persyaratan Sistem
  • PHP: Versi 8.1 atau lebih tinggi
  • MySQL: Versi 8.0 atau MariaDB 10.5+
  • RAM: Minimum 2GB (4GB direkomendasikan)
  • Storage: 10GB untuk aplikasi dan data
  • Browser: Chrome 90+, Firefox 88+, Safari 14+
Perintah Instalasi
# Clone repository git clone https://github.com/sutiyonodoang/kesehatanremajaV2.git # Install dependencies composer install npm install # Setup environment cp .env.example .env php artisan key:generate # Setup database php artisan migrate php artisan db:seed # Build assets npm run build php artisan storage:link # Start development server php artisan serve
